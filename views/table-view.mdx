## Overview

Table View presents your trades in a spreadsheet-style format, perfect for quickly scanning multiple trades, comparing data, and getting a high-level overview of your trading activity.

## Accessing Table View

1. Navigate to the **Trades** section
2. Click the **Table** view toggle in the top navigation
3. Your trades appear in rows with data columns

## Table Columns

The table displays key trade information:

| Column | Description |
|--------|-------------|
| **Title** | Trade name/description |
| **Symbol** | Ticker symbol |
| **Date** | Trade date |
| **Direction** | Long or Short |
| **Result** | Win, Loss, Breakeven, Open |
| **Rating** | 1-5 star rating |
| **Status** | Workflow status |
| **Tags** | Applied tags |

## Sorting

Click any column header to sort:
- **First click** - Sort ascending (A-Z, oldest first, etc.)
- **Second click** - Sort descending (Z-A, newest first, etc.)
- **Third click** - Remove sort

<Tip>
  The current sort column shows an arrow indicator (↑ or ↓).
</Tip>

## Row Actions

Click any row to:
- Open the trade detail view
- View full notes and images
- Edit trade information

## When to Use Table View

Table View is ideal for:

- **Quick scanning** - Review many trades at a glance
- **Data comparison** - Compare values across trades
- **Bulk review** - Process multiple trades efficiently
- **Finding specific trades** - Combined with filters

## Compared to Other Views

| View | Best For |
|------|----------|
| **List** | Visual browsing with thumbnails |
| **Board** | Workflow management |
| **Table** | Data-focused scanning |
